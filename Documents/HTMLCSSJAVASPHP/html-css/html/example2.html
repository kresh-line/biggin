Τρόπος προβολής Επικεφαλίδων

Ο τρόπος προβολής των επικεφαλίδων από τους φυλλομετρητές είναι τέτοιος ώστε να φαίνονται μεγαλύτερες οι επικεφαλίδες που έχουν μικρότερο αριθμό από αυτές που έχουν μικρότερο και ουσιαστικά είναι υποενότητές τους. Φυσικά αυτό δεν σημαίνει ότι δεν πρέπει να τις μορφοποιούμε με CSS όπως θα δούμε αργότερα καθώς ο κάθε φυλλομετρητής μπορεί να έχει δικό τους τρόπο προβολής.
